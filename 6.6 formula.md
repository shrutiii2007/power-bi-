## lab work 6.6 formulas 

| Questions | Formulas |
|-----------|-------------|
| Q1 | Expensive Product Sales = 
CALCULATE(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    FILTER(
        Products,
        Products[Price] > 100
    )
) |
| Q2 | High Volume Customers = 
COUNTROWS(
    FILTER(
        VALUES(Sales[CustomerID]),
        CALCULATE(COUNTROWS(Sales)) > 5
    )
) |
| Q3 | Weighted Average Price = 
DIVIDE(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    SUM(Sales[Quantity])
) |
| Q4 | Total Discount Amount = 
SUMX(
    Sales,
    Sales[Quantity] * RELATED(Products[Discount])
) |
| Q5 | Max Revenue Per Invoice = 
MAXX(
    Sales,
    Sales[Quantity] * RELATED(Products[Price])
) |
| Q6 | YTD Sales = 
TOTALYTD(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    'Calendar'[Date]
) |
| Q7 | QTD Sales = 
TOTALQTD(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    'Calendar'[Date]
) |
