## 📊 Lab Work 6.6 – DAX Formulas

| Questions | Formulas |
|----------|----------|
| Q1 | ```DAX
Expensive Product Sales =
CALCULATE(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    FILTER(
        Products,
        Products[Price] > 100
    )
)
``` |
| Q2 | ```DAX
High Volume Customers =
COUNTROWS(
    FILTER(
        VALUES(Sales[CustomerID]),
        CALCULATE(COUNTROWS(Sales)) > 5
    )
)
``` |
| Q3 | ```DAX
Weighted Average Price =
DIVIDE(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    SUM(Sales[Quantity])
)
``` |
| Q4 | ```DAX
Total Discount Amount =
SUMX(
    Sales,
    Sales[Quantity] * RELATED(Products[Discount])
)
``` |
| Q5 | ```DAX
Max Revenue Per Invoice =
MAXX(
    Sales,
    Sales[Quantity] * RELATED(Products[Price])
)
``` |
| Q6 | ```DAX
YTD Sales =
TOTALYTD(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    'Calendar'[Date]
)
``` |
| Q7 | ```DAX
QTD Sales =
TOTALQTD(
    SUMX(
        Sales,
        Sales[Quantity] * RELATED(Products[Price])
    ),
    'Calendar'[Date]
)
``` |
